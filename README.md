
# ML-Stock-Market-Index-Analysis

The model and code are used to analyze and forecast stock indices using the LSTM network. The model uses historical data to capture temporal relationships and generate predictions of future values.

# Functions:
 - data loading and preprocessing,
 - LSTM model training,
 - forecasting and result visualization

# Project can be run:
- locally 
- in a web-based environment, such as Google Colab
